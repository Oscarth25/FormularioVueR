# Actividad 9 – Formulario interactivo con Vue.js

**Unidad V: Frameworks de JavaScript**  
**Curso:** LR-20261-INF-4314-LR02

---

## 📋 Descripción

Aplicación web desarrollada con **Vue 3 + Vite** que implementa un formulario interactivo para capturar datos de estudiantes y los muestra dinámicamente en una tabla dentro de la misma página.

## 🛠 Tecnologías

- **Vue 3** (Composition API con `<script setup>`)
- **Vite** como bundler / servidor de desarrollo
- CSS nativo con variables personalizadas (sin librerías externas)
